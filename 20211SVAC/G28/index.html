ESTRUCTURAS DE DATOS

Visualiza estructuras de datos de forma dinámica para comprender su funcionamiento de una forma mil veces más fácil

Estructuras lineales

Lista simple enlazada

Una lista enlazada simple es una estructura de datos en la que cada elemento estructura de datos en la que cada elemento apunta al siguiente.

Lista doble enlazada

Tipo de lista enlazada que permite ir en ambas direcciones, hacia adelante y hacia atrás, en una lista enlazada. .

Lista circular simple

Tiene el mismo tipo de nodos que una lista simple enlazada... Consiste en reemplazar el valor NULL del nexo del último nodo, por la dirección del nodo cabeza de la lista

Lista circular doble

Cada nodo tiene dos enlaces, similares a los de la lista doblemente enlazada, excepto que el enlace anterior del primer nodo apunta al último y el enlace siguiente del último nodo, apunta al primero

Pila

Una pila es una lista ordenada o estructura de datos que permite almacenar y recuperar datos, siendo el modo de acceso a sus elementos de tipo LIFO.

Cola

Una cola es una estructura de datos, caracterizada por ser una secuencia de elementos en la que la operación de inserción push se realiza por un extremo y la operación de extracción pull por el otro.

Cola de prioridad

Una cola de prioridades es un tipo de dato abstracto similar a una cola en la que los elementos tienen adicionalmente, una prioridad asignada.​​

Ordenamientos

Burbuja

Funciona revisando cada elemento de la lista que va a ser ordenada con el siguiente, intercambiándolos de posición si están en el orden equivocado

Selección

Consiste en encontrar el menor de todos los elementos del vector e intercambiarlo con el que está en la primera posición.

Inserción

El ordenamiento por inserción es una manera muy natural de ordenar para un ser humano, y puede usarse fácilmente para ordenar un mazo de cartas numeradas en forma arbitraria.

Rápido

El ordenamiento rápido usa dividir y conquistar para obtener las mismas ventajas que el ordenamiento por mezcla, pero sin utilizar almacenamiento adicional.

Estructuras arbóreas

Árbol binario de búsqueda

Todos los elementos almacenados en el subárbol izquierdo de cualquier nodo son menores que el elemento almacenado en el lado derecho y viceversa

AVL

Es un árbol binario de búsqueda que satisface la condición de estar balanceado.

Árbol B

son estructuras de datos de árbol que se encuentran comúnmente en las implementaciones de bases de datos y sistemas de archivos.

Árbol B+

Es un tipo de estructura de datos de árbol, representa una colección de datos ordenados de manera que se permite una inserción y borrado eficientes de elementos.

Árbol de Merkle

Estructura de datos en árbol, binario o no, en el que cada nodo que no es una hoja está etiquetado con el hash de la concatenación de las etiquetas o valores de sus nodos hijo.
